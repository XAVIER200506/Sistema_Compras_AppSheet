# Avance semana 10

En esta carpeta se almacenan las evidencias y documentos correspondientes al avance de la semana 10 del proyecto **Sistema_Compras_AppSheet**.

## Módulos funcionales desarrollados

- Solicitudes de compra
- Detalle de solicitud
- Aprobación y rechazo de solicitudes
- Órdenes de compra
- Detalle de orden de compra
- Recepción
- Detalle de recepción
- Evaluación
- Gestión documental
- Automatización básica por correo

## Evidencias sugeridas para esta carpeta

- Capturas del módulo de solicitudes
- Capturas del módulo de órdenes
- Capturas del módulo de recepción
- Capturas del módulo de evaluación
- Capturas de gestión documental
- Captura de la automatización por correo
- Guía o documento de sustentación de semana 10

## Descripción general del avance

Para esta etapa del proyecto ya se cuenta con un flujo funcional del proceso de compras dentro de AppSheet. El sistema permite registrar solicitudes, agregar productos, aprobar o rechazar solicitudes, generar órdenes de compra, registrar la recepción, evaluar el proceso y adjuntar documentos soporte.

Adicionalmente, se implementó una automatización básica que envía un correo cuando una solicitud cambia al estado **Enviada**.

## Observación

Este avance corresponde a la fase de desarrollo y validación funcional del sistema. Como siguiente etapa se puede fortalecer con indicadores, reportes, controles por rol y nuevas automatizaciones.
