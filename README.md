# Sistema de Gestión de Compras

Proyecto desarrollado como práctica empresarial utilizando AppSheet y Google Sheets.

## Objetivo
Diseñar e implementar un sistema para la gestión de solicitudes de compra, proveedores y control de procesos de adquisición.

## Módulos implementados
- Usuarios
- Roles
- Proveedores
- Solicitud de compra
- Detalle de solicitud

## Funcionalidades actuales
- Creación de solicitudes de compra
- Asociación de usuarios solicitantes
- Registro de productos en la solicitud
- Cálculo automático de subtotales
- Cálculo automático del monto total

## Tecnologías utilizadas
- AppSheet
- Google Sheets
